
# Anthony Hernandez – Portfolio

Deployed at https://www.anthony-hernandez.net via GitHub Pages.

**Structure**
```
/assets
index.html
404.html
CNAME
README.md
```

**Deploy steps**
1. Upload all files to a GitHub repository named `anthonyhernandez`.
2. In GitHub → Settings → Pages:
   - Source: `main` / `(root)`
   - Custom domain: `www.anthony-hernandez.net`
   - Enforce HTTPS: ✅ ON
3. Visit the site: https://www.anthony-hernandez.net
